# The Heera Architects

A responsive architecture and interior design portfolio built with React, Vite, Tailwind CSS, Framer Motion, and Lucide React.

## Run locally

```bash
npm install
npm run dev
```

## Brand logo

The supplied brand artwork is stored unchanged at `public/logo.png` and is used
in the header and footer.

## Editing content

Update services, projects, process steps, and contact information in `src/data/siteData.js`.
